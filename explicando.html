Introdução

Este documento fornece uma visão geral das funções e comportamentos utilizados do código JS. O código implementa um jogo da velha - TRABALHO SENAC TECH RS - em uma página da web.

Variáveis Globais

  • currentPlayer: Uma referência ao elemento HTML que exibe qual jogador deve fazer o próximo movimento.
  • selected: Um array que armazena as seleções dos jogadores no tabuleiro.
  • player: Uma string indicando qual jogador deve fazer o próximo movimento.
  • positions: Uma matriz que define todas as combinações vencedoras possíveis no jogo da velha.

Funções

init()

Descrição: Inicializa o jogo da velha.

Comportamento:

  • Define selected como um array de 9 elementos, inicialmente preenchido com null.
  • Atualiza o texto do elemento currentPlayer para indicar qual jogador deve fazer o próximo movimento.
  • Remove todos os eventos de clique dos botões do jogo e os limpa.
  • Adiciona um evento de clique a cada botão do jogo, ligando-os à função newMove().
  • As três primeiras submatrizes, representam as três linhas do tabuleiro do jogo, onde cada número representa a posição correspondente na linha.
  • As próximas três submatrizes representam as três colunas do tabuleiro do jogo.
  • Das duas últimas submatrizes, representam as duas diagonais do tabuleiro do jogo.
  • E, então dessas combinações, são usadas na função check(). Verifica se um jogador conseguiu preencher todas as posições em uma dessas combinações. Isso indica que "ele" venceu o joguinho

newMove(e)

Descrição: Lida com o "evento" de clique em um botão do jogo.

Parâmetros:

  • e: O evento de clique.

Comportamento:

  • Obtém o índice do botão clicado.
  • Atualiza o texto do botão clicado com o símbolo do jogador atual.
  • Remove o evento de clique do botão clicado.
  • Registra a seleção do jogador no array selected.
  • Aguarda 200ms e, em seguida, verifica se houve um vencedor ou empate.
  • Altera o jogador atual.
  • Atualiza o texto do elemento currentPlayer para refletir o próximo jogador.

check()

Descrição: Verifica se há um vencedor ou empate no jogo.

Comportamento:

  • Itera sobre todas as posições vencedoras possíveis.
  • Verifica se o jogador atual preencheu todas as posições de uma combinação vencedora.
  • Se um jogador venceu, exibe um alerta indicando o vencedor e reinicia o jogo.
  • Se não houver vencedor e todos os espaços estiverem preenchidos, exibe um alerta de empate e reinicia o jogo.

Execução Inicial

A função init() é chamada inicialmente para iniciar o jogo da velha.
